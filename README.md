# Aurum

## Description

Aurum is a fine dining restaurant website designed to provide an elegant online experience for potential customers. The site showcases the restaurant's premium offerings, including an exquisite menu, reservation system, and stunning visual gallery. 

## Features

- **Hero Section**: Landing page with call-to-action buttons for reservations and menu exploration.
- **About Section**: Detailed information about the restaurant's passion for perfection, including years of excellence, number of dishes, Michelin stars, and customer satisfaction.
- **Menu**: Interactive display of signature dishes, including appetizers, mains, desserts, and beverages with pricing.
- **Gallery**: Visual showcase of the restaurant's ambiance and dishes.
- **Reservations**: User-friendly form for booking tables, including date, time, number of guests, and special requests.
- **Newsletter Signup**: Subscription form for updates on seasonal menus, events, and tastings.
- **Contact & Footer**: Complete contact information, opening hours, and social media links.
